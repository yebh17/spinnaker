About Spinnaker

Spinnaker is a open source multi cloud continuous delivery platform that helps to release software changes.

Developed Netflix. It enables teams to automate deployments across multiple cloud accounts and regions even accross multiple cloud platforms, into coherent pipelines that run forever whenever a new version in released. Netflix deployed its first microservice to the cloud in 2009.
